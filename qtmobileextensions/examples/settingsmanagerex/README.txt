Settings Manager example

Application description:
This application is for testing the Settings Manager API.

Features:
- Shows the current device profile
- Shows the status of the charger.
- Shows basic usage of the Publish and Subscribe mechanism.

Tested platforms:
- Qt 4.5.2-tower
- S60 3rd edition FP2 SDK v1.1
- S60 5th edition SDK v1.0

Phones tested with:
- N82
- N95
- N96
- 5800 XpressMusic
