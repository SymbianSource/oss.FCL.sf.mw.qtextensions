SysInfo

Application description:
This application is for testing the SysInfo API.

Features:
- Retrieve information about the phone (model, imei, language, etc.).
- Get information about the supported/unsupported features in the device.

Tested platforms:
- Qt 4.5.2-tower
- S60 3rd edition FP2 SDK v1.1
- S60 5th edition SDK v1.0

Phones tested with:
- N82
- N95
- N96
- 5800 XpressMusic
